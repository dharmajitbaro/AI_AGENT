# 🤖 Gorq AI Assistant

A smart AI assistant built with LangGraph and Groq that can search the web, fetch weather data, and remember conversations.

---

## 🚀 Features
- Web search (DuckDuckGo)
- Real-time weather info
- Conversation memory
- Powered by Groq LLaMA 3.3

---

pip install langgraph langchain-groq langchain-community langchain-core requests ddgs
